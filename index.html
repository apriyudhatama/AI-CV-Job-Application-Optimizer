API Settings

Configure your Google Gemini API Key to enable real AI-powered optimization. Without an API Key, the application runs in local simulation mode.

Your API key is stored safely in your browser (localStorage) and never transmitted to our servers.

Privacy & Usage

100% Client-Side: All data parsing, keyword audits, editing, and exports occur entirely within your web browser. No resume text is stored on external databases.

Getting a Key: You can obtain a free Gemini API Key from Google AI Studio. The free tier includes high rate limits, perfect for personal resume refinement.

Offline Simulation: If you don't configure a key, our built-in NLP parser scans and highlights common resume pitfalls, injecting relevant role-specific skills.
